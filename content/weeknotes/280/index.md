  * Gonna keep this short because I'm writing over lunch and I feel like not a lot happened.

  * Currently migrating all my Open Source projects to drop support for Node.js 20 which is nicely repetitive and mind-numbing. I'm also switching all my little side projects over to TypeScript now that Node.js supports [type stripping](https://nodejs.org/api/typescript.html#type-stripping). It's really nice, I love being able to run TypeScript code without a build process – it simplifies things so nicely. We've been doing the same at work when we can justify the little bit of extra tech debt time.

  * Writing a lot of Terraform at work and I feel like I've learned a lot. More confident with writing modules and I've managed to tidy up _some_ of the mess made by Claude previously. There's far less copy/paste and the documentation is more complete. I wouldn't really know but, based on the code I've seen so far, using an agent makes it extremely easy to just park your brain for a while. Must be relaxing.
