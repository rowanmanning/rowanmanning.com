
# Rowan's Website

The source code for [rowanmanning.com](https://rowanmanning.com/).


## Running locally

  1. Install [Hugo](https://gohugo.io/)
  2. Run `hugo server`
  3. Visit [http://localhost:1313/](http://localhost:1313/)
